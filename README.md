Smart City Management System (SQL Project)

📌 Project Overview

The Smart City Management System is a database project designed to manage various services and operations within a smart city.
This project was developed using SQL to organize and analyze city-related data such as citizens, transport, complaints, utilities, and emergency services.

🎯 Objectives

- Manage citizen information efficiently
- Track public complaints and services
- Monitor transport and traffic data
- Maintain records of hospitals, schools, and emergency services
- Improve city administration through structured data management

🛠 Technologies Used

- SQL
- MySQL Database
- Database Design Concepts

🗂 Database Structure

The project contains multiple tables to manage different smart city services.

Example tables:

- Citizens
- Services
- Complaints
- Transport
- Traffic Fines
- Hospitals
- Schools
- Water Supply
- Electricity Supply
- Waste Management
- Parking
- Emergency Services
- Events

⚙ Features

- Well-structured relational database
- Primary Key and Foreign Key relationships
- Realistic dataset for analysis
- Multiple SQL queries for data retrieval and management

📊 Sample SQL Operations

- Data Insertion
- Data Filtering
- JOIN Operations
- Aggregation Queries
- Scenario-based queries for analysis

🚀 How to Use

1. Create the database in MySQL
2. Run the table creation scripts
3. Insert the dataset into tables
4. Execute SQL queries for analysis and insights
